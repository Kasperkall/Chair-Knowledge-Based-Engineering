To run in Windows, execute the runnable batch file 'runOlingvo.bat'.

To run on Linux or Mac, execute the runnable shell script 'runOlingvo.sh'.

On all systems, Olingvo can be executed by running the Olingvo.jar file. For example, on Windows, you would run the command 'java -jar Olingvo.jar' in the command prompt in the directory containing Olingvo.
